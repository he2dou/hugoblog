git 是一种版本控制系统，它可以帮助开发人员管理代码，记录代码变更历史，提高团队协作效率。

<!--more-->

{{< toc >}}


## 安装

- windows上安装

> 使用官方地址进行下载, https://git-scm.com/downloads

- centos上安装

```sh
yum install -y git
```

## 操作


```sh
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
```
